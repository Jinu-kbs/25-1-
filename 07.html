Prompt Writing Techniques

프롬프트의 개념

프롬프트(Prompt)는 AI 모델에게 주어지는 입력 텍스트로, AI가 어떻게 응답할지 결정하는 지침이 됩니다.

좋은 프롬프트의 특징

  • 명확하고 구체적인 지시 포함
  • 원하는 출력 형식 명시
  • 충분한 맥락 정보 제공
  • 단계별 지시 활용

효과적인 프롬프트 기법

역할 부여

"화학 교육 전문가로서, 이 실험을 설명해주세요"

단계별 지시

"다음 단계로 나누어 실험 과정을 안내해주세요: 1. 준비물, 2. 실험 방법, 3. 결과 분석"

출력 형식 지정

"마크다운 표를 사용하여 실험 결과를 정리해주세요"

반복적 개선

"이 응답을 더 간결하게 만들고 중학생이 이해할 수 있도록 수정해주세요"

실험 시뮬레이션 프롬프트 예시

핵융합 반응 시뮬레이션

당신은 과학 교육 전문가입니다. 핵융합 반응에 대한 시뮬레이션을 만들어 주세요. 다음 내용을 포함해주세요:

  1. 1. 별의 중심부에서 일어나는 수소 핵융합 과정을 단계별로 설명
  2. 2. 중수소와 삼중수소가 결합하여 헬륨이 되는 과정 시각화
  3. 3. 핵융합 반응에서 발생하는 에너지 계산 예시
  4. 4. 중학생들이 이해할 수 있는 수준으로 설명
상세한 단계별 핵융합 과정 시뮬레이션 생성

불꽃반응 실험 시뮬레이션

화학 선생님 역할을 맡아, 불꽃반응 실험의 가상 시뮬레이션을 제공해주세요:

  1. 1. 다양한 금속염(Na, K, Cu, Ca, Sr)의 불꽃색 설명
  2. 2. 각 금속이 특정 색을 나타내는 원리(전자 전이와 파장)
  3. 3. 학생들이 관찰할 수 있는 현상 상세 설명
  4. 4. 실험 과정 중 예상되는 결과와 사진 묘사
  5. 5. 학생 질문에 대비한 QnA 포함
금속별 불꽃색과 원리가 포함된 시뮬레이션

프롬프트 최적화 팁

  • 학습 목표를 명확히 제시하세요
  • 학생 수준(중학생/고등학생)을 명시하세요
  • 시각적 묘사를 요청하여 이해도를 높이세요
  • 결과물을 단계별로 구조화하도록 요청하세요
